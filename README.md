# Number Row Characters

Here are the characters you get when pressing `Shift` along with the numbers on the top row of the keyboard:

- `!` (exclamation mark)
- `@` (at symbol)
- `#` (hash or pound)
- `$` (dollar sign)
- `%` (percent sign)
- `^` (caret)
- `&` (ampersand)
- `*` (asterisk)
- `(` (left parenthesis)
- `)` (right parenthesis)
